# KEYLOGGERS
Keylogger is a Python-based project designed to capture and log keystrokes. It demonstrates how key-logging can be used for various purposes, such as monitoring typing activity for research, troubleshooting keyboard issues, or learning how keyloggers work for cybersecurity education.

📌 Keylogger Project (Educational)

A simple Python-based keylogger built using pynput and tkinter to demonstrate how keyboard input can be captured and logged in real-time.

🚀 Features
Real-time keystroke logging
JSON + text log output
Simple GUI (Start/Stop)
Lightweight and beginner-friendly
⚠️ Disclaimer

This project is created for educational and ethical cybersecurity learning purposes only. Do not use it without proper authorization.

🛠️ Tech Used
Python
Tkinter
Pynput
